Task 2: 
Function to Calculate Factorial
Objective: Write a recursive function that calculates the factorial of a number.
Instructions:

Define a function named factorial that takes one parameter.
The function should return the factorial of the number using recursion.
Call the function with different numbers and print the results.

Task 3: 
Function to Generate Fibonacci Sequence
Objective: Write a function that generates the first n numbers in the Fibonacci sequence.
Instructions:

Define a function named fibonacci that takes one parameter.
The function should return an array containing the first n Fibonacci numbers.
Call the function with different values of n and print the results.
